# ApplicationStatusOverlay

The ApplicationStatusOverlay is used to render a Status View within the application.

When mounted, the ApplicationStatusOverlay will communicate with an ancestor
[ApplicationStatusOverlayProvider](/application/cerner-terra-application/components/application-status-overlay-provider) to
present a staus view.

> See the [How to Show Status Overlay View](/application/cerner-terra-application/how-to/show-status-overlays) page for usage information.

## Usage

```jsx
import ApplicationStatusOverlay from '@cerner/terra-application/lib/application-status-overlay';
```
